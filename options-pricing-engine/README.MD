## What Does This Project Do?

Options pricing is a fundamental problem in quantitative finance â€” given a set of market inputs, what is the fair price of an option contract today?

This project solves that problem using two methods:

- **Black-Scholes** â€” a closed-form mathematical formula that gives an instant theoretical price
- **Monte Carlo Simulation** â€” simulates thousands of possible future stock price paths, calculates the payoff for each, and averages them to estimate the price

The app is fully web-based with interactive sliders and inputs â€” no coding or installation required to use it.
